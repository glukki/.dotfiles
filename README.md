dotfiles
==============
Configuration files for everything. Use [stow](https://www.gnu.org/software/stow/stow.html) to symlink them fast.

Install
-------
```
cd ~
git clone git@github.com:glukki/.dotfiles.git
cd .dotfiles

# this will install stow
./bin/,provision-mac

make link
```
